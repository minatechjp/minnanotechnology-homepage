# minnanotechnology-homepage

Static GitHub Pages site for みんなのテクノロジー合同会社 / MINNA NO TECHNOLOGY LLC.

## Files

- `index.html` - homepage
- `company.css` - styles
- `company.js` - language switcher and link behavior
- `company-hero.png` - hero image
- `.nojekyll` - disables Jekyll processing on GitHub Pages

## Publish with GitHub Pages

1. Create a GitHub repository named `minnanotechnology-homepage`.
2. Upload these files to the repository root.
3. Open repository `Settings` -> `Pages`.
4. Select `Deploy from a branch`.
5. Select branch `main` and folder `/root`.

The published URL will be similar to:

`https://minatechjp.github.io/minnanotechnology-homepage/`
